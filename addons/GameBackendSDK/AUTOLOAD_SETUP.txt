# Autoload Setup Instructions

## Option 1: Via Godot Editor (Recommended)

1. Open your project in Godot
2. Go to Project â†’ Project Settings
3. Click on the "Autoload" tab
4. Click the folder icon next to "Path"
5. Navigate to and select: res://addons/GameBackendSDK/Backend.gd
6. Set "Node Name" to: Backend
7. Click "Add"
8. Close Project Settings

## Option 2: Manual project.godot Edit

Add this to your project.godot file under the [autoload] section:

[autoload]

Backend="*res://addons/GameBackendSDK/Backend.gd"

## Verification

To verify the autoload is working, add this to any script:

func _ready() -> void:
    print("Backend available: ", Backend != null)
    var result := await Backend.init("https://api.example.com", "test_project")
    print("Init result: ", result)

You should see:
- "Backend available: true"
- Init result showing ok=true if configured correctly
